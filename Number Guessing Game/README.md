# 🎯 Number Guessing Game (GUI)

A simple Python GUI game built using CustomTkinter where the user guesses a random number between 1 and 1000.


https://github.com/user-attachments/assets/8c514527-d3a9-4054-b2fa-53bc311a358f


## 🚀 Features
- Clean GUI using CustomTkinter
- Input validation (only integers allowed)
- Smart hints (too high / too low / slightly close)
- Attempt counter (max 30 attempts)
- Performance feedback:
  - 🌟 Excellent (1–5 attempts)
  - 🔥 Great (6–20 attempts)
  - 👍 Good (21–30 attempts)
- Game ends after 30 attempts or correct guess

## 🛠️ Tech Stack
- Python
- CustomTkinter

## ▶️ How to Run
```bash
pip install customtkinter
python main.py
